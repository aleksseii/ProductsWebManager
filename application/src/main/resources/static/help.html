HELP

To see this help message [No Role, Guest or Manager]:

GET http://localhost:3466


To get list of all products [Guest or Manager]:

GET http://localhost:3466/product/all


To get list of all products produced by company with provided name [Guest or Manager]:

GET http://localhost:3466/product/all/{string}

(provide name of company instead of `{string}` of corresponding type)


To add new product [Manager]:

POST http://localhost:3466/product/add

provide characteristics of new product in the request's body with JSON like { "productName" : {string}, "companyName" : {string}, "amount" : {integer} }

(provide product characteristics instead of `{string}`, `{integer}` of corresponding types)


To delete products with provided name [Manager]:

POST http://localhost:3466/product/delete/{string}

(provide product name instead of `{string}` of corresponding type)
